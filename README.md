# Hi, I'm Samuel 👋

**Electromechanical Systems Engineer** based in Hong Kong.

I build integrated hardware products where mechanical architecture and electronic subsystems are co-designed from the ground up — from SolidWorks CNC-ready structures to custom carrier boards, embedded Linux bring-up, and vision pipelines.

---

## 🔧 What I Work On

**Mechanical Design**
- Mechanism synthesis — spring-latch clamping, gear/rack actuation, six-bar linkages
- CNC-ready SolidWorks models with GD&T, tolerance stack-up, and DFM coordination
- Structural design for electromechanical products and sensor-integrated enclosures

**Embedded Hardware & PCB**
- Custom carrier board design for RK3588S (LubanCat) and Intel N150 (LattePanda Mu) SoMs
- High-speed PCB layout — USB3, PCIe, MIPI CSI-2 (Altium Designer / EasyEDA)
- Power architecture — USB-C PD negotiation, LiHV battery management, multi-rail sequencing
- FPGA development — Lattice CrossLink-NX, MIPI-to-UVC pipeline

**Embedded Linux & Bring-up**
- Hardware bring-up and fault isolation across power, I2C/PCIe bus, and DTS configuration
- Embedded Linux (Ubuntu / Buildroot), DTS overlay authoring, sysfs/GPIO/PWM control

**Computer Vision & Sensing**
- Stereo camera synchronization (hardware VSYNC via PWM)
- Human pose estimation — OpenCV, MediaPipe, YOLOv8
- IMU-based motion tracking with Vicon ground-truth calibration
